# Connector Logos

This directory contains logos for API connectors displayed in the marketplace.

## Adding Logos

You can either:

1. **Download actual logos** from the service provider websites (recommended for production)
2. **Use placeholder images** for development:
   - Stripe: https://via.placeholder.com/48x48/5B21B6/FFFFFF?text=S
   - SendGrid: https://via.placeholder.com/48x48/1A73E8/FFFFFF?text=SG
   - Slack: https://via.placeholder.com/48x48/4A154B/FFFFFF?text=SL
   - Australia Post: https://via.placeholder.com/48x48/DC143C/FFFFFF?text=AP
   - Xero: https://via.placeholder.com/48x48/13B5EA/FFFFFF?text=X

## File Naming Convention

- `stripe.png` or `stripe.svg`
- `sendgrid.png` or `sendgrid.svg`
- `slack.png` or `slack.svg`
- `australiapost.png` or `australiapost.svg`
- `xero.png` or `xero.svg`

## Recommended Size

- 48x48 pixels or larger (will be scaled down in the UI)
- PNG or SVG format
